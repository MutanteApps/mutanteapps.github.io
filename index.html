N-Queens

In chess games, there is a 8x8 board, 16 pieces of each side, with One Queen. This piece can move as many cells as the player wants in any direction.
But in this app, we change the rules: There will be a NxN board and N Queens, but they can take each other! So this app calculates possible positions where each Queen must be to be safe.
